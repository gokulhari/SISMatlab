We would love to hear from you if you have used SISMatlab! Get in touch with Mihailo Jovanovic or Gokul Hariharan with your comments and suggestions.

If you run into bugs with SISMatlab, file a bug report here and we'll to get back to you. If possible, please include the following information:

  1. Error messages produced by Matlab

  2. Matlab version used

  3. Platform (Linux/Windows/Mac)

  4. Matlab code that can reproduce the problem

If you have ideas to make SISMatlab better, report them here. Your help is greatly appreciated.
